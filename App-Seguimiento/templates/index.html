Bienvenidos a la Base de Datos de SeguimientoAPP




Instrucciones para cargar un medio:

  • De acuerdo al tipo de reproductor -Youtube, M3u8 o Radios- ingrese a la pestaña de carga
  • En cada formulario, ingresar todos los datos requeridos
  • Están disponibles las opciones para modificar datos o eliminar un medio


Links de transmisión

  • En Youtube, se ingresa el VideoID
  • Si el link en vivo del video es el siguiente: https://www.youtube.com/watch?v=cb12KmMMDJA, el VideoID es "cb12KmMMDJA"
  • En M3u8, se ingresa el link completo, por ejemplo: https://stream1.sersat.com/hls/garagetv.m3u8
  • En Radios, se ingresa el link completo, por ejemplo: https://sa.mp3.icecast.magma.edge-access.net/sc_rad1
  • Recordar que en la categoría "Reproductor" se coloca el nombre del tipo de audio: mp3, aac o hls -medios m3u8-


Instrucciones para datos sobre un medio:

  • Ingrese a la pestaña "Otros Datos"
  • Buscar el medio y editar todos los datos requeridos
